# Оптична система виявлення лазерного
опромінення

Суть проєкту: Прототип захисної системи для техніки, що попереджає екіпаж про наведення
лазерного прицілу чи далекоміра

1. Якщо фоторезистор показує більше 3000, вмикаємо тривогу.
2. При натисканні на кнопку можна вимкнути тривогу або перейти в режим маскування (не виявляемо лазер).
